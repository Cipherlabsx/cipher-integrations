2. DefiLlama

DL

DefiLlama — Volume & Fees

PR Ready

Dimension-adapter reporting daily volume and fees. Fetches from the Orbit DEX adapter API.

Submit to
github.com/DefiLlama/dimension-adapters
File path
dexs/orbit-finance/index.ts
Methodology
Aggregate 24h swap volume across all pools, convert to USD via quote token prices. Fees = volume × pool fee rate.

Data Flow

GET /api/v1/pools           → pool IDs, quote mints, fee rates
GET /api/v1/volumes?tf=24h  → 24h volume per pool (quote-denominated)
GET /api/v1/tokens/prices   → quote token USD prices

dailyVolume = Σ(volumeQuote × quotePriceUsd)
dailyFees   = Σ(volumeUsd × baseFeeBps / 10000)
